# SoraYama's personal blog site

> Powered by [Gatsby](https://www.gatsbyjs.org/) and [fabien's typescript gatsby starter](https://github.com/fabien0102/gatsby-starter)

## Commands

```bash
# Install
yarn

# Develop
yarn start

# Deploy
yarn test
yarn build

# Generate template
yarn generate

# Generate GraphQL typedefs
yarn graphql-types

# storybook
yarn storybook
```
